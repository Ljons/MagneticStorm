# Магнітні бурі (MagneticStorm)

Android-додаток для перегляду **рівня магнітних бурь** (Kp-індекс) на сьогодні та прогнозу на кілька днів.

## Можливості

- **Поточний рівень** — Kp-індекс на сьогодні з позначкою шкали NOAA (G1–G5).
- **Прогноз на кілька днів** — по днях з максимумом Kp та інтервалами по 3 години.
- **Вибір локації** — введіть місто та країну (наприклад, «Київ, Україна»). Дані відображаються в вашому часовому поясі; вибір зберігається після закриття додатку.
- **Світла та темна тема** — підтримка світлої, темної та теми «як у системи» (Налаштування).
- **Сучасний інтерфейс** — Material Design 3, Compose.

## Джерело даних

- **NOAA Space Weather Prediction Center** — безкоштовний API для Kp-індексу та прогнозу:  
  https://services.swpc.noaa.gov/products/
- **Nominatim (OpenStreetMap)** — геокодування для пошуку міст і визначення часового поясу за країною.

## Як зібрати

1. Відкрийте проєкт в **Android Studio** (Hedgehog або новіший).
2. Дозвольте синхронізацію Gradle (якщо потрібно — буде завантажено `gradle-wrapper.jar`).
3. Запустіть на емуляторі або пристрої: **Run → Run 'app'**.

Або з терміналу (потрібні JDK 17 та [Gradle](https://gradle.org/install/)):

```bash
cd MagneticStorm
./gradlew assembleDebug
```

APK: `app/build/outputs/apk/debug/app-debug.apk`.

## Структура проєкту

- `app/src/main/java/.../data/` — API (NOAA, Nominatim), моделі, репозиторії, DataStore.
- `app/src/main/java/.../ui/` — екрани (Compose), тема, ViewModel.
- Локація та тема зберігаються через **DataStore**.

## Ідеї для подальшого розвитку

- Сповіщення при високому Kp (наприклад, G2 і вище).
- Віджет на головний екран з поточним Kp.
- Коротка довідка: що таке Kp, як він впливає на здоров’я та техніку.
- Історія Kp за минулі дні (графік).
- Підтримка англійської та інших мов (зараз інтерфейс українською).

## Ліцензія

Проєкт використовує відкриті API NOAA та OSM Nominatim; дані надаються згідно з їхніми умовами використання.
